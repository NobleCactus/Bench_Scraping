In order for this script to work properly, the following needs to be in place:
	- Git repo cloned, dependencies installed
	- Chromedriver must be installed (https://chromedriver.storage.googleapis.com/index.html?path=109.0.5414.74/)
	- In the Documents folder in Windows, another folder named 'DPN Profiles' must be created

From there, the script can be executed.
